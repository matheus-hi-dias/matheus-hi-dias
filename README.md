# 👋 Hi! I'm Matheus Dias

### 💻 About Me
I'm a **Fullstack Developer** specializing in web and mobile development. Currently, I work at **Multiclínica Mais Saúde**, where I build applications using **Next.js** and **NestJS**, implementing both monolithic and microservices architectures.

I focus on building scalable, high-performance applications, always prioritizing **Clean Code**, **TDD/BDD**, and modern development practices.
